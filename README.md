Santan Solar — Documentation & Resources
This repository serves as the official customer resource library for Santan Solar. All product datasheets, installation guides, wiring diagrams, and technical reference materials are published here and accessible via GitHub Pages.
Whether you're a homeowner reviewing product specifications before purchase or an installer setting up a system on-site, you'll find everything you need here.
What's included:

Product datasheets (panels, inverters, batteries, and accessories)
Step-by-step installation guides
Wiring and system diagrams
Warranty and compliance documentation

All documents are kept up to date with the latest product revisions. If you have questions or can't find what you're looking for, please reach out to us at support@santansolar.com.
